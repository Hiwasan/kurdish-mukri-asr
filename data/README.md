# Data Structure

This project uses Kurdish Mukri audio data for ASR training.

## Expected Data Format:

### Audio Files:
- Location: `data/audio/`
- Format: WAV, 16kHz mono
- Naming: Unique identifiers

### Transcriptions:
- Location: `data/transcriptions/`  
- Format: TSV files with columns: audio_id, transcription

## Data Access:
The actual Kurdish Mukri dataset is not included in this repository for privacy reasons. Researchers interested in accessing the data should contact the repository maintainer directly.

## For Reproduction:
To use this code with your own data, organize your files according to the structure above and update the configuration files accordingly.
